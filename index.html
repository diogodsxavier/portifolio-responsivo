Sobre mim

Desenvolvedor Front-End

Minha trajetória na programação começou com a curiosidade sobre como os sites e aplicações são construídos, o que me levou a desenvolver habilidades tanto no front-end quanto no back-end. No front-end, tenho conhecimento com HTML, CSS e JavaScript, criando interfaces de usuário intuitivas e responsivas. No back-end, meus conhecimentos é com Node.js e Express. Também possuo conhecimento em MongoDB para gerenciamento de dados, garantindo um fluxo de informação eficiente e seguro. Familiarizado com controle de versão usando Git e GitHub, ferramentas que considero essenciais para qualquer desenvolvedor moderno. Valorizo imensamente o trabalho em equipe e acredito que a comunicação eficaz é fundamental para o sucesso de qualquer projeto. Tenho foco em resolução de problemas, sempre buscando soluções criativas para os desafios que surgem.
